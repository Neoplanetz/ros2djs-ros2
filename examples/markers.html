Simple MarkerArray Example

Run rosbridge then publish a few markers (ROS 2):

  1. ros2 launch rosbridge_server rosbridge_websocket_launch.xml
  2. Publish a CUBE marker at (1, 2): 
    ros2 topic pub --once /markers visualization_msgs/msg/MarkerArray "{markers: [{header: {frame_id: 'map'}, ns: 'demo', id: 1, type: 1, action: 0, pose: {position: {x: 1.0, y: 2.0, z: 0.0}, orientation: {w: 1.0}}, scale: {x: 0.5, y: 0.5, z: 0.5}, color: {r: 1.0, g: 0.0, b: 0.0, a: 1.0}}]}"
  3. DELETEALL to clear: 
    ros2 topic pub --once /markers visualization_msgs/msg/MarkerArray "{markers: [{action: 3}]}"
